Individual Project Site Plan

Site Name

Studio One Portfolio

This name represents a clean and professional personal portfolio website for showcasing software development projects. “Studio One” also aligns with my home studio branding, giving the site a creative and modern identity.

Optional domain availability: studioone-dev.com

Site Purpose

The purpose of this website is to present a personal portfolio that highlights my software development skills, projects, and experience. It will provide project case studies, a professional bio, contact information, and links to GitHub and LinkedIn. The goal is to attract clients and potential employers.

Scenarios

  • What projects has this developer completed, and what technologies were used?
  • How can I contact this developer for freelance or collaboration opportunities?

Color Scheme

Primary Color: #0B5CFF
Used for: headings, buttons, highlights, borders

Accent Color: #111827
Used for: navigation text, footer background

Background Color: #F3F4F6
Used for: full-page background and section contrast

Typography

Poppins (Headings)
Used for: all major headings (H1–H3) to create a clean modern look.

Roboto (Body Text)
Used for: paragraphs, navigation items, footer text.
